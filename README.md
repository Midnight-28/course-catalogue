# Course Catalogue

### Development

-   Fork Repository

-   Clone Repository

```sh
git clone https://github.com/{Your_Username}/course-catalogue.git
cd course-catalogue
```

-   Install Backend Dependencies

```sh
npm install
```

-   Install Frontend Dependencies

```sh
npm run client-install
```

-   Run Local MongoDB Server

-   Run Development Server

```sh
npm run dev
```
